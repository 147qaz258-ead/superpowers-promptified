# Superpowers for Cursor

Superpowers workflow automation plugin for Cursor AI IDE.

## Installation

### Via Cursor Settings
1. Open Cursor Settings
2. Navigate to Plugins
3. Add marketplace: `https://github.com/obsidianmd/obsidian-publish/tree/master`
4. Install superpowers

## Features

- Brainstorming - Design before implementation
- Planning - Structured implementation plans
- Code review - Quality checks
- Git workflows - Isolated development

## Configuration

Configure skills and commands in Cursor settings to customize your workflow.
