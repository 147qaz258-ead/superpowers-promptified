# Superpowers for Coder

Superpowers workflow automation plugin for Coder AI IDE.

## Installation

### Via Coder Settings
1. Open Coder Settings
2. Navigate to Extensions
3. Search for "superpowers"
4. Install

## Features

- Brainstorming - Design before implementation
- Planning - Structured implementation plans
- Code review - Quality checks
- Git workflows - Isolated development

## Configuration

Configure skills and commands in Coder settings to customize your workflow.
