# Brainstorm Command

Invokes the brainstorming skill to explore requirements and design before implementation.

## Usage

```bash
/brainstorm
```

This will:
1. Ask clarifying questions about the task
2. Explore multiple approaches
3. Present design for approval
4. Only proceed to implementation after approval

## When to Use

- Starting a new feature
- Modifying existing behavior
- Adding new functionality
- Architectural decisions

## Related Skills

- writing-plans - Creates implementation plan after brainstorming
- executing-plans - Executes the implementation plan
