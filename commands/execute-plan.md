# Execute Plan Command

Invokes the executing-plans skill to implement a written plan.

## Usage

```bash
/execute-plan
```

This will:
1. Load the plan file
2. Review it critically
3. Execute tasks in batches
4. Report progress between batches
5. Wait for feedback

## When to Use

- You have a written plan from writing-plans
- Ready to implement features
- Following structured development workflow

## Related Skills

- writing-plans - Creates the plan to execute
- finishing-a-development-branch - Completes the work
